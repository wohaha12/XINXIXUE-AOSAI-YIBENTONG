# 题目标题：[1003: 对齐输出]

## 时间限制: 1000 ms   内存限制: 66536 KB

## 题目难度：⭐

## 算法标签：[输入输出、格式化输出、C++ 基础]

## 【题目描述】

读入三个整数，按每个整数占8个字符的宽度，右对齐输出它们，按照格式要求依次输出三个整数，之间以一个空格分开。

## 【输入】

只有一行，包含三个整数，整数之间以一个空格分开。

## 【输出】

只有一行，按照格式要求依次输出三个整数，之间以一个空格分开。

## 【输入样例】

```text
123456789 0 -1
```

## 【输出样例】

```text
123456789      0       -1
```

## 🧠 解题思路

1. 步骤一：通过C++标准输入流（cin）读取三个整数，存储到变量中。
2. 步骤二：使用格式化输出，设置每个整数占8个字符宽度且右对齐（利用`setw`和`right`操纵符）。
3. 步骤三：按顺序输出三个整数，中间用空格分隔。

## 📝 总结

1. **`setw(n)`：设置输出字段宽度**  
   - 作用：指定输出的最小宽度为 `n` 个字符。若输出内容长度小于 `n`，会在前面（或后面，取决于对齐方式）填充空格，确保总宽度达到 `n`。  
   - 示例：`setw(8)` 表示每个输出项至少占8个字符宽度，不足部分用空格填充。  
2. **`right`：设置右对齐方式**  
   - 作用：将输出内容在指定字段宽度内**右对齐**，即内容靠右，前面用空格填充剩余宽度。  
   - 示例：`cout << setw(8) << right << a` 表示整数 `a` 在8个字符宽度的区域内右对齐，前面用空格填充不足的部分。  
3. **`<iomanip>` 头文件**  
   - 需包含 `<iomanip>` 头文件才能使用 `setw`、`right` 等流操纵符，用于控制输出格式。
