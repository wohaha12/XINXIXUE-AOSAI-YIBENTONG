# 题目标题：[1004: 字符三角形]

## 时间限制: 1000 ms   内存限制: 66536 KB

## 题目难度：⭐

## 算法标签：[输出、格式化输出、C++ 基础]

## 【题目描述】

给定一个字符，用它构造一个底边长5个字符，高3个字符的等腰字符三角形。

## 【输入】

输入只有一行，包含一个字符。

## 【输出】

该字符构成的等腰三角形，底边长5个字符，高3个字符。

## 【输入样例】

```text
*
```

## 【输出样例】

```text
  *
 ***
*****
```

## 🧠 解题思路

1. 步骤一：通过C++标准输入流（cin）读取一个字符。
2. 步骤二：构造等腰三角形，第1行输出1个字符（居中，前面2个空格），第2行输出3个字符（居中，前面1个空格），第3行输出5个字符（前面0个空格）。
3. 步骤三：按顺序输出三行，确保字符居中显示。
