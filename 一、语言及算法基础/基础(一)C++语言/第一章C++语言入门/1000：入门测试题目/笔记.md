# 题目标题：[1000：入门测试题目]

## 时间限制: 1000 ms   内存限制: 32768 KB

## 题目难度：⭐

## 算法标签：[输入输出、基础运算、C++ 基础]

## 【题目描述】

输入两个整数，输出它们的和。

## 【输入】

一行，两个用空格隔开的整数

## 【输出】

两个整数的和。

## 【输入样例】

```text
2 3
```

## 【输出样例】

```text
5
```

## 🧠 解题思路

1. 步骤一：通过C++标准输入流（cin）读取两个整数，存储到变量中。
2. 步骤二：对两个整数执行加法运算，计算求和结果。
3. 步骤三：通过C++标准输出流（cout）输出求和结果。
